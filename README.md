# My-Lesson

My-Lesson 是一个基于 Spring Cloud 微服务架构的在线课程与营销管理平台，配套提供管理员后台前端，用于统一管理用户、课程、营销活动与订单数据。

## 项目架构

- 微服务后端：按业务拆分为用户、课程、订单、营销等服务，网关统一入口。
- 管理员前端：`ml-web` 提供后台管理界面，支持动态菜单与权限校验。

## 模块与职责

- `ml-gateway`：统一网关入口，路由转发与跨服务访问控制。
- `ml-user`：用户/角色/菜单管理，用户角色/角色菜单关系，登录鉴权相关接口。
- `ml-course`：课程、分类、季次、集次、评论、举报、关注等课程域管理。
- `ml-order`：订单与订单明细管理，订单统计与支付相关接口。
- `ml-sale`：营销内容与活动管理（公告、轮播图、文章、秒杀、秒杀详情、优惠券等）。
- `ml-common`：公共工具与基础设施（通用返回、分页结构、校验规则、工具类等）。
- `ml-web`：管理员后台前端（Vite + Vue + Element Plus）。

## 主要功能

- 统一登录与权限校验
- 动态菜单与页面路由
- 管理端列表数据展示与分页
- 课程/用户/营销/订单等后台管理能力
- 管理端数据概览与统计展示（配合可视化组件）

## 配置与路由

- 配置中心：Nacos
- 服务路由：Gateway 统一入口转发
